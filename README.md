## About

Etch-a-sketch that allows you to choose the color and grid size

## How to Play

Pick a color and grid size, click on the grid to draw

## How to View

Go to Orbit274.github.io/etch-a-sketch

## Files

*   `index.html`: Front page
*   `script.js`: Logic for drawing, erasing, picking colors, and changing grid size
*   `styles.css`: Formatting of page and changes in color and style
